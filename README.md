# Admin Portal - React.js

This is an Admin Portal built using React.js and Vite. It includes multiple pages like:

- Dashboard
- Users
- Orders
- Products
- Reports
- Notifications
- Settings
- Support
- Profile
- Login

## 🛠 Tech Stack
- React.js
- React Router DOM
- Bootstrap (UI components)
- Vite

## 📁 Folder Structure
src/

│ ├── Users.jsx
│ ├── Orders.jsx
│ └── ...
